## Step 2: Projection Mapping

### PART 2: Projection mapping.

  

In order to demonstrate the principle of projection mapping, I slightly modified the code from¬†[Method draw](https://github.com/methodofaction/Method-Draw), a tool developed by¬†*[Mark MacKay](https://method.ac/)*. It works in any computer and browser.

  

[https://joanielemercier.com/mapping/](https://joanielemercier.com/mapping/)

  

**Instructions:**

**1- Setup your projector**

1. Install and plug projector (it must be stable and NOT MOVE at all).
2. Make sure the projection area covers the origami.
3. Focus image.
4. Connect your computer (you may use duplicate or extend mode).

![](https://i0.wp.com/joanielemercier.com/wp-content/uploads/2020/03/giphy.gif?resize=480%2C270&ssl=1)

  

**2- Open¬†[the mapping tool](https://joanielemercier.com/mapping/)**¬†in a browser (I use chrome) in full screen mode (F11).

On a Mac, it‚Äôs ‚ÄúApple + ctrl + F‚Äù.

  

Select the canvas size on the right, to match your projector resolution (often 1920*1080).

That‚Äôs it, you‚Äôre all set !

![](https://i0.wp.com/joanielemercier.com/wp-content/uploads/2020/04/giphy.gif?resize=480%2C270&ssl=1)

**3- Trace the edges**

Select the¬†**Line tool (shortcut: L)**.

Place your cursor on a corner,

Click and drag to the next corner. Repeat.

  

Use¬†**Select (V)**¬†to select and move elements.

You can save your progress as an .svg file.

Repeat until the entire structure is mapped.

  

![](https://i0.wp.com/joanielemercier.com/wp-content/uploads/2020/04/click_trace.gif?resize=480%2C270&ssl=1)

**4- Animation mode**

When all lines are traced, you can switch to the animation mode:

2. Press¬†**‚ÄúN‚Äù**¬†to turn the ‚Äònoise animation‚Äô on. It‚Äôs a generative mask that hides some of the lines.
